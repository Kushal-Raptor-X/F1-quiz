# F1 Quiz Game 🏎️

A console-based Formula 1 quiz game with 10 multiple choice questions.

## How to Run

```bash
python "F1 Quiz.py"
```

## How to Play

1. Type 'Y' to start
2. Answer questions by typing 1-4
3. Type 'Y' to continue after each question
4. Type 'R' to restart or any key to exit

## Features

- 10 F1 questions covering drivers, teams, and history
- Score tracking
- Restart option

## Requirements

- Python 3.x
